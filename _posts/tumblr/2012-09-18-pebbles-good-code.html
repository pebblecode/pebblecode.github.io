pebble {code}’s experienced and friendly team of software developers and designers, is offering one month of free service to a charitable / not-for-profit organisation. Whether you have a problem you need to solve, or want to explore a fresh idea, if we agree that it can be done in one month, we’ll have a go!

First and foremost, we want to build a reputation for producing first class software. We’re also committed to doing so in a socially responsible manner, and by contributing to the open source community.

what is open source?

Open source software is software that anyone can use or modify. We have used software from many open source projects in the past, and want to give back to that community.

what can we do?

pebble {code}’s main expertise lies in our ability to solve complex problems, although there is a limit to the amount of complexity that we can build in one month with one developer and one designer. Our main available resources at the moment are in our web department. Some realistic project examples that could feasibly be tackled and solved in one month could be:

  • A web-based database to track donors
  • An online survey component to your website
  • A funky data visualisation based on your sector
  • A web application prototype that tracks projects and contacts
  • A browser based game

applications

Please fill out the simple web form, at the bottom of this page, by midday 12th October 2012. Successful applicants will be notified by email shortly after the closing date and we will set up interviews via Skype (or face to face if in London).

We’re aiming to start and complete a one month project before the end of this year.

We look forward to hearing from you.
